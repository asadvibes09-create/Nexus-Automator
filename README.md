# Nexus Automator – Intelligent Desktop Automation Suite 🚀

Nexus Automator is a high-performance desktop productivity suite built with Python and CustomTkinter. It leverages rule-based Natural Language Processing (NLP) and multi-threading to automate repetitive file management and data extraction tasks locally and securely.

---

## 🌟 Key Features

* **🧠 Smart AI File Categorizer:** Safely parses content inside `.pdf` and `.docx` files using text scoring engines, automatically organizing them into structured hubs (*Education, Tech, Finance, Legal*).
* **🌗 Dynamic UI/UX Theme Switcher:** Features a premium dark and light mode toggle with an asynchronous, fully responsive layout built entirely on CustomTkinter.
* **⚡ Multi-Threaded Bulk Renamer:** Processes hundreds of files simultaneously with real-time progress indicators, eliminating application freezing.
* **📊 Targeted Web Scraper:** Extracts unstructured web data and formats it directly into structured Excel files for quick analysis.
* **🔒 100% Offline & Private:** Runs entirely on the local machine with zero external cloud API costs, ensuring complete data privacy.

---

## 🛠️ Tech Stack

* **Language:** Python 3.x
* **GUI Framework:** CustomTkinter (Asynchronous Layouts)
* **Text Parsing & NLP:** PyPDF2, python-docx
* **Data Processing:** Pandas, OpenPyXL
* **Automation & Concurrency:** Threading, Git
